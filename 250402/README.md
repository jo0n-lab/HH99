여기서 확인해보면 좋을 사항은, 출력이 되는 값 [x1, y1, x2, y2] 에 대해 x1, x2 를 구하는 방법이 조금 다를 수 있다는 거?

이렇게 다른 이유는 **파일** 이 실제로는 그림과 같이 **1과 2 사이** 에 있기 때문이다. 
따라서 1개의 파일이라 하더라도 직사각형의 크기는 1이다. 0이 될 수 없다는 말이다.
